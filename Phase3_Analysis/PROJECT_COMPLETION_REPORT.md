## 🚨 已知限制与改进空间

### 限制1: 论文范围
- **现状:** 仅覆盖2025-2026的Tier I/II/III论文
- **影响:** 2024及之前的基础工作可能遗漏
- **建议:** 定期更新以纳入新论文

### 限制2: 定量数据来源
- **现状:** 直接采用论文报告的数字，未二次验证
- **影响:** 可能存在论文间的不一致性 (e.g., 准确率定义差异)
- **建议:** 后续可进行跨论文的数据标准化

### 限制3: 应用场景覆盖
- **现状:** 主要聚焦云系统和微服务RCA
- **影响:** 医疗、金融等领域的特殊需求可能遗漏
- **建议:** 建立领域专项的扩展分析

### 限制4: 因果关系的深度
- **现状:** 主要是主题-论文的对应，缺乏论文间的深度引用关系图
- **影响:** 无法直观看出论文间的理论继承关系
- **建议:** 构建文献依赖图 (citation network)
